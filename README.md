## TEAMPARK

TeamPark is an augmented video platform for remote working. TeamPark utilized WebRTC streams for sharing video and screens.

# Development

Start server:
```
npm run start-server
```

Start client:
```
npm run watch
```

Open:
```
http://localhost:5474/$room-id
```
